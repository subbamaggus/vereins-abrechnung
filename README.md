# vereins-abrechnung

Das Projekt implementiert Abrechnung für kleinere Vereine.

## geplante Features

* Multimandant
* html Interface auch für Mobile Geräte nutzbar
* Konto/Kassenstand Gegenprüfung
* Passwörter werden per POST übertragen > SSL
* Multilevel Rechteverwaltung
* Export Ready für Elster
* Export in pdf
* Foto von Quittungen
* Audit Trail, versehentliches Bearbeiten kann nachvollzogen werden
